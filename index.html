# Melis Bulut — Landingpage Statische Website (HTML/CSS/JS, keine Frameworks). Direkt auf GitHub Pages deploybar — einfach den Repo-Inhalt pushen und in den Repo-Einstellungen unter "Pages" aktivieren. `index.html` enthält CSS und JS bereits eingebettet (eine einzige Datei). Bilder liegen als separate Dateien daneben. ## Bereits eingebunden - **Hero-Foto**: `assets/images/hero.jpg` - **Porträt**: `portrait.jpg` (im selben Verzeichnis wie `index.html`) - **Live-Auftritt (S/W)**: `assets/images/performance-bw.jpg` — für die Web-Nutzung auf 1000px Breite verkleinert und komprimiert (Original war 4016×6016px, das wäre für eine Website unnötig groß und würde die Ladezeit verschlechtern) - **Testimonials**: drei echte Zitate sind eingebaut (ohne Namen, da keine Zuordnung vorlag — bei Bedarf ergänzen) ## Vor dem Launch noch zu erledigen 1. **WhatsApp-Nummer**: Aktuell überall `491234567890` als Platzhalter (Suchen & Ersetzen in `index.html`). 2. **Audio-Sample**: `assets/audio/sample.mp3` fehlt noch — ein kurzer Live-Ausschnitt zum Anhören. 3. **OG-Bild**: `assets/images/og-cover.jpg` für die Social-Media-Vorschau fehlt noch. 4. **Instagram-Link** im Footer auf das echte Profil anpassen. 5. **Impressum**: Der Footer-Link verweist aktuell nur auf einen Anker (`#impressum`) mit einem Minimal-Copyright-Hinweis. Für den rechtssicheren Betrieb in Deutschland sollte ein vollständiges Impressum (Name, Anschrift, Kontakt) ergänzt werden. 6. **Testimonial-Zuordnung**: Falls ihr möchtet, dass zu den drei Zitaten ein Name/Ort steht — sagt mir, welche, dann baue ich das ein. ## Struktur ``` index.html (CSS + JS bereits eingebettet) portrait.jpg assets/ images/ hero.jpg performance-bw.jpg icons/ audio/ (noch anzulegen für sample.mp3) ```
