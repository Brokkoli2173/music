# Melis Bulut — Landingpage

Statische Website (HTML/CSS/JS, keine Frameworks). Direkt auf GitHub Pages deploybar — einfach den Repo-Inhalt pushen und in den Repo-Einstellungen unter "Pages" aktivieren.

## Vor dem Launch unbedingt ersetzen

1. **WhatsApp-Nummer**: Aktuell überall `491234567890` als Platzhalter (Suchen & Ersetzen in `index.html`).
2. **Bilder** (in `assets/images/`):
   - `hero.jpg` — Hero-Hintergrund (Foto/Video-Standbild einer Performance)
   - `proof.jpg` — emotionaler Beweis-Moment (z.B. erster Tanz)
   - `portrait.jpg` — warmes Porträt von Melis als Person
   - `og-cover.jpg` — Vorschaubild für Social-Media-Links
   - Aktuell sind das CSS-Farbverläufe als Platzhalter — echte Fotos sind für die Premium-Wirkung entscheidend.
3. **Audio-Sample**: `assets/audio/sample.mp3` — ein kurzer Live-Ausschnitt (aktuell fehlt die Datei, der Player zeigt dann einen Hinweistext).
4. **Testimonials**: Die drei Zitate im Testimonials-Bereich sind Platzhalter — durch echte Kundenstimmen ersetzen.
5. **Instagram-Link** im Footer auf das echte Profil anpassen.
6. **Impressum**: Der Footer-Link verweist aktuell nur auf einen Anker (`#impressum`) unten auf derselben Seite mit einem Minimal-Copyright-Hinweis. Für den rechtssicheren Betrieb in Deutschland sollte ein vollständiges Impressum (Name, Anschrift, Kontakt) ergänzt werden.
